# كيفية تثبيت المشروع
1. تأكد من تثبيت `Node.js` و `npm` على جهازك.
2. قم بتنزيل الكود من GitHub:
   ```bash
   git clone https://github.com/Motaz3d/av.git
   ```
3. انتقل إلى مجلد المشروع:
   ```bash
   cd av
   ```
4. قم بتثبيت الاعتمادات:
   ```bash
   npm install
   ```
5. قم بتشغيل المشروع:
   ```bash
   npm start
   ```
